# Project Brutality Extended - Core
This is the Main Repository for Project Brutality Extended

------------


## Description
Project Brutality Extended is an addon for the [Doom (1993)](https://store.steampowered.com/app/2280/DOOM__DOOM_II/) Mod [Project Brutality](https://github.com/pa1nki113r/Project_Brutality/tree/PB_Staging). It is a series of addon that adds everything from weapons, items, armors, and many quality of life features. The goal for this mod is quality above everything else so I try to make everything in zscript and integrate every feature in PB so it comes as close to the original mod standards as possible.

But in doing so means progress for these mods will be very slow as I am only working alone.

The Repository you are looking at now is the Core and the hub for all PBX Modules. The Core itself can be run as a standalone mod but is *required* for every other PBX Modules.

The Core contains functions and libraries needed for other PBX Modules to run, in terms of gameplay this module adds new HUD elements such as current weapon and armor used.

## How to Download
1. First you will need the latest version of [UZDoom/LZDoom](https://zdoom.org/downloads) for PC, [Delta Touch](https://play.google.com/store/apps/details?id=com.opentouchgaming.deltatouch&hl=en_US&gl=US "Delta Touch") for Android, or [GenZD](https://apps.apple.com/us/app/genzd/id6503916449 "GenZD") for iOS (as of July 7th 2026 the latest version is 4.14.3)
2. Next you will need to obtain a copy of  either a copy of the one of the official IWADs <sup>[(1)][1] [(2)][2]</sup> or [Freedoom Phase 1+2](https://freedoom.github.io/download.html "Freedoom").
3. Then grab a copy of [Project Brutality](https://github.com/pa1nki113r/Project_Brutality/tree/PB_Staging) by opening the link, scroll up to the top and click the green button labeled __Code__ then click [Download Zip](https://github.com/pa1nki113r/Project_Brutality/archive/refs/heads/PB_Staging.zip "You can also click here to download."). If you are on mobile and do not see a button, you will need to turn on desktop view.
4. After that you will need to download PBX itself by doing the same thing with PB by scrolling up and clicking the green button labeled __Code__ then click Download Zip
5. Last step is to load every mod using a Mod Loader such as [AceCorp](https://gitlab.com/accensi/acecorp-launcher/-/releases)

A couple of things to note:
1. You do not need to change the format of the file (.Zip) to PK3, you can just load the file as is
2. *Always* load PBX - Core first *before* any other modules, the rest of the order does not matter as long as PBX - Core is loaded first
